# Stingy Shopper

A smart market board shopping list manager and path optimizer plugin for **Final Fantasy XIV** (Dalamud). 

Stingy Shopper helps you find the cheapest servers to buy items on your shopping list, optimizes your travel routes using **Lifestream**, and automatically tracks your purchases in real-time.

---

## Features

### 🛒 Dynamic Optimization Modes
Stingy Shopper provides three customizable optimization algorithms depending on how much you want to travel to save gil:
- **Maximum Stingy (Default)**: Splits your shopping list across servers to secure the absolute lowest price for every single item.
- **Balanced**: Groups items by server, but only splits items to other worlds if the savings exceed a customizable threshold percentage (e.g. 10% cheaper).
- **Maximum Convenience**: Analyzes the total cost of your entire list across all worlds and recommends buying everything in bulk on the single server that offers the cheapest overall price.

### ⚡ Chained Lifestream Integration
- **One-Click Teleportation**: Initiates world travel directly from the shopping plan UI.
- **Chained Routing**: World travel commands automatically chain with Lifestream's action queue to teleport you directly to the nearest market board once you land on the target server.
- **Zone-Aware Navigation**: Dynamically displays a "Go to Market Board" button when you are already on the current world, letting you warp to the local market board instantly.

### 📋 Seamless List Importers
- **Teamcraft Clipboard Importer**: Paste exported lists directly from your clipboard. Filter imports by category (crystals, timed nodes, pre-crafts, gathering, tomes/scrips, dungeon drops, etc.).
- **MakePlace Shopping List Importer**: Load `.json` shopping lists generated by MakePlace to quickly source housing items.

### 🔍 Automated Purchase Tracking
- Hooks into chat logs using **SeString ItemPayload parsing** to track purchase confirmations in real-time. 
- Auto-decrements item counts and reorganizes your shopping plan dynamically.
- Highly robust: immune to pluralizations (e.g., `"bottles of Windspath water"`), language differences, and formatting symbols.

## Installation

1. Open `/xlsettings` in game.
2. Navigate to the **Experimental** tab.
3. Add the following repository URL to your custom plugin repositories list:
   ```
   https://raw.githubusercontent.com/KaraRemy/TomokPlugins/main/pluginmaster.json
   ```
4. Search for **Stingy Shopper** in the plugin installer and install it.

---

## Configuration Settings
Stingy Shopper's configuration window allows you to customize:
- **Stinginess Mode**: Select between Maximum Stingy, Balanced, or Maximum Convenience.
- **Balanced Savings Threshold (%)**: Define the minimum savings required to justify splitting an item to another world.
- **Track Purchases from Chat**: Toggle real-time automatic shopping list decrementing.
- **Auto-Copy Item Name on Click**: Click any item name in the plan to instantly copy it to your clipboard for quick pasting into the market board.
- **Region-Wide Lookup**: Scan prices across the entire region rather than just your local Data Center.
- **Cross-DC Travel Limit (%)**: Threshold to restrict travel between Data Centers unless savings are significant.
